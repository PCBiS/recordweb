치지직 쿠키 추출 방법:

1. 웹 브라우저에서 네이버 페이지를 열고 로그인합니다.
2. F12 → Application 탭 → Cookies에서 https://www.naver.com 또는 https://chzzk.naver.com을 선택합니다.
3. NID_AUT, NID_SES 값을 복사하여 입력합니다.
4. 치지직 연령제한 방송은 해당 계정 쿠키가 있어야 합니다.

씨미 쿠키 추출 방법:

1. 웹 브라우저에서 씨미(https://ci.me)에 로그인합니다.
2. F12 → Application 탭 → Cookies → https://ci.me를 선택합니다.
3. mauth-authorization-code, session-id 값을 복사하여 입력합니다.
4. 씨미 연령제한/4K 구독 방송은 해당 계정 쿠키가 있어야 합니다.
